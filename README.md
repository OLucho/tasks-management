# Car Rental Agency
<p align="center">Car rental agency System development. Backend made with Nest, Frontend with React</p>


### Tech Stack

- 🟥**NestJs** —  Progressive Node.js Framework
- 📄 **TypeORM** — A Promise-Based Node.js ORM
- 🟦 **TypeScript** — An open-source language which builds on JavaScript by adding static type definitions.
- ⚛️ **React Js** — A JavaScript library for building user interfaces
- ♻️ **Hooks & Context** — To handle State
- 🌀 **React Router Dom** — DOM bindings for React Router.
- 🐘 **PostgreSQL** — A powerful RDBMS
- ⚙️ **Cypress** - Testing E2E
- ⚙️ **Jest** - Unit Testing
